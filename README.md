## 🚀 Features

### 🔊 Audio Pronunciation
- Hover over any vocabulary card to hear the German word pronounced
- Uses native German voice (de-DE) via Web Speech API
- Adjustable speed for clear learning

### 🖼️ Image-Based Memory
- Tap any card to open a full modal with real-world images
- Automatically fetches images from Wikipedia & Wikimedia Commons
- "Next Image" button to cycle through multiple images

### 💡 AI Hinglish Explanation (Claude AI)
- Click "Explain (Hinglish)" on any word's modal
- Gets a real-life, casual Hinglish explanation using Claude AI
- Includes a practical German sentence + memory trick
- "Read Aloud" feature reads the explanation in a natural voice

### 🔍 Smart Search
- Search across German, English, and Hindi simultaneously
- Instant filtering with live results count

### 🏷️ Category Tabs
- 18 different vocabulary categories
- Color-coded badges for quick visual identification

## 🏃 How to Run Locally

No installation needed! Just open the file:

```bash
git clone https://github.com/iikram42/German-Vocab.git
cd German-Vocab
# Open index.html in your browser
start index.html        # Windows
open index.html         # Mac
```

> **Note:** The AI Hinglish explanation feature requires a Claude API key in the source code.

## 💡 How to Use

1. **Browse** — cards load with Today's Vocab by default
2. **Hover** — hold mouse over a card to hear pronunciation
3. **Tap/Click** — open the modal for image + full details
4. **Explain** — click "Explain (Hinglish)" for a casual story-based explanation
5. **Search** — type in German, English, or Hindi to find any word
6. **Filter** — click category tabs to focus on one topic
